# PWA Installation Guide - Mazar App
# دليل تثبيت تطبيق مزار

## English Version

### What is a PWA?

A Progressive Web App (PWA) is a website that works like a native app. Once installed:
- ✅ Launches from your home screen
- ✅ Works offline
- ✅ Faster performance
- ✅ No app store required
- ✅ Always up-to-date

### Installation Instructions

#### Chrome (Android & Desktop)

1. **Visit** https://your-app-url.com
2. **Look for** the install icon in the address bar
3. **Click** the install button
4. **Confirm** installation
5. **Done!** App added to your home screen/applications

**Or:**
- Tap the menu (⋮)
- Select "Install app" or "Add to Home Screen"
- Tap "Install"

#### Safari (iPhone/iPad)

1. **Visit** https://your-app-url.com
2. **Tap** the Share button (□↑) at the bottom
3. **Scroll down** and tap "Add to Home Screen"
4. **Edit** the name if desired
5. **Tap** "Add"
6. **Done!** App icon added to your home screen

**Tips:**
- The share button is at the bottom center of Safari
- If you don't see "Add to Home Screen", scroll down in the share menu

#### Safari (Mac)

1. **Visit** https://your-app-url.com
2. **Click** File menu
3. **Select** "Add to Dock"

**Or:**
- Click the share button in toolbar
- Select "Add to Dock"

#### Edge (Desktop)

1. **Visit** https://your-app-url.com
2. **Click** the install icon (⊕) in the address bar
3. **Click** "Install"
4. **Done!** App opens in its own window

#### Firefox

Firefox doesn't support automatic PWA installation, but you can:
- Bookmark the site for quick access
- Create a desktop shortcut

## النسخة العربية

### ما هو تطبيق PWA؟

تطبيق الويب التقدمي (PWA) هو موقع ويب يعمل مثل التطبيق العادي. بعد التثبيت:
- ✅ يُفتح من الشاشة الرئيسية
- ✅ يعمل بدون إنترنت
- ✅ أداء أسرع
- ✅ لا يحتاج متجر تطبيقات
- ✅ يُحدّث تلقائياً

### تعليمات التثبيت

#### Chrome (أندرويد وسطح المكتب)

1. **زُر** https://your-app-url.com
2. **ابحث عن** أيقونة التثبيت في شريط العنوان
3. **اضغط** على زر التثبيت
4. **أكّد** التثبيت
5. **تم!** أُضيف التطبيق إلى شاشتك الرئيسية

**أو:**
- اضغط على القائمة (⋮)
- اختر "تثبيت التطبيق" أو "إضافة إلى الشاشة الرئيسية"
- اضغط "تثبيت"

#### Safari (آيفون/آيباد)

1. **زُر** https://your-app-url.com
2. **اضغط** على زر المشاركة (□↑) في الأسفل
3. **مرر للأسفل** واضغط "إضافة إلى الشاشة الرئيسية"
4. **عدّل** الاسم إذا أردت
5. **اضغط** "إضافة"
6. **تم!** أُضيفت أيقونة التطبيق إلى شاشتك الرئيسية

**نصائح:**
- زر المشاركة في منتصف الأسفل في Safari
- إذا لم تجد "إضافة إلى الشاشة الرئيسية"، مرر للأسفل في قائمة المشاركة

#### Safari (ماك)

1. **زُر** https://your-app-url.com
2. **اضغط** على قائمة File
3. **اختر** "Add to Dock"

**أو:**
- اضغط على زر المشاركة في شريط الأدوات
- اختر "Add to Dock"

#### Edge (سطح المكتب)

1. **زُر** https://your-app-url.com
2. **اضغط** على أيقونة التثبيت (⊕) في شريط العنوان
3. **اضغط** "تثبيت"
4. **تم!** يُفتح التطبيق في نافذة منفصلة

#### Firefox

Firefox لا يدعم التثبيت التلقائي، لكن يمكنك:
- حفظ الموقع في المفضلة للوصول السريع
- إنشاء اختصار على سطح المكتب

## Benefits / المزايا

### Fast Access / وصول سريع
Launch directly from your home screen or applications folder
افتح التطبيق مباشرة من الشاشة الرئيسية أو مجلد التطبيقات

View cached content even without internet connection
اطّلع على المحتوى المحفوظ حتى بدون اتصال بالإنترنت

### Native Experience / تجربة مثل التطبيقات العادية
Full screen experience without browser UI
تجربة ملء الشاشة بدون واجهة المتصفح

### Automatic Updates / تحديثات تلقائية
Always get the latest features automatically
احصل دائماً على أحدث المزايا تلقائياً

### Better Performance / أداء أفضل
Faster load times and smoother interactions
أوقات تحميل أسرع وتفاعلات أكثر سلاسة

## Troubleshooting / حل المشكلات

### Install Button Not Showing / زر التثبيت لا يظهر

**Possible Reasons:**
- App is already installed
- Browser doesn't support PWA
- Site must be on HTTPS

**الأسباب المحتملة:**
- التطبيق مثبت بالفعل
- المتصفح لا يدعم PWA
- الموقع يجب أن يكون على HTTPS

**Solution:**
1. Check if app is already installed
2. Try a different browser (Chrome recommended)
3. Clear browser cache and try again

**الحل:**
1. تحقق من أن التطبيق غير مثبت بالفعل
2. جرب متصفح آخر (Chrome مستحسن)
3. امسح ذاكرة المتصفح المؤقتة وحاول مرة أخرى

### App Not Opening / التطبيق لا يُفتح

**Solution:**
1. Uninstall and reinstall the app
2. Check internet connection
3. Clear app cache

**الحل:**
1. احذف التطبيق وثبته مرة أخرى
2. تحقق من اتصال الإنترنت
3. امسح ذاكرة التطبيق المؤقتة

### Uninstalling / إلغاء التثبيت

**iPhone/iPad:**
- Long press the app icon
- Tap "Remove App"
- Tap "Delete App"

**Android:**
- Long press the app icon
- Tap "App info" or drag to "Uninstall"
- Confirm removal

**Desktop:**
- Open the app
- Click menu (⋮)
- Select "Uninstall"
